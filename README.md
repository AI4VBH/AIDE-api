## Description

The backend API for AIDE.

<!-- INSTALLATION -->
## Installation

- Install [Node & NPM](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm).

```bash
$ npm install
```

<!-- USAGE -->
## Usage

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
